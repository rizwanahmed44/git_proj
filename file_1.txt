Steps to create repository in git and github

1. Goto working directory in git
	
	$ cd <directory path/name>

2. Check the working directory

	$ pwd

3. See the list of files

	$ ls -la

4. How to see list of files

	$ ls

5. How to check if there is anything to commit

	$ git status

6. How to tracked file

	$ git add <file name>
	$ git add -A  (add all files)
